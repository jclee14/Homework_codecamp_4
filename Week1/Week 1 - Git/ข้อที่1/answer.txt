Question1: หน้าที่ของ Branch คืออะไร

Answer: Branch คือฟังก์ชั่นที่ช่วยให้ developer สามารถพัฒนาหรือแก้ไขโค้ดได้ โดยไม่กระทบกับโค้ดต้นฉบับ
รวมถึงเอื้ออำนวยการพัฒนาโค้ดชุดเดียวกันจาก developer หลาย ๆ คนอย่างเป็นระบบ (สมาชิกสามารถแยกพัฒนากันได้)
และแก้ไข conflict จากการ merge ได้อย่างมีประสิทธิภาพ // หลักการพื้นฐานคือการ duplicate โค้ดต้นฉบับ
แล้วแยกมาเป็น branch ใหม่ สำหรับใช้พัฒนาตามจุดประสงค์ของ branch นั้น ๆ เมื่อโค้ดถูกพัฒนาเสร็จสิ้นแล้ว
developer จะ merge branch ที่ต้องการกลับเข้าไปรวมกลับโค้ดต้นฉบับ (จะเห็นได้ว่าตอนที่กำลังพัฒนาโค้ดใน branch
โค้ดต้นฉบับ รวมถึงbranchอื่น ๆ ที่ถูกพัฒนาโดย developer ท่านอื่นจะไม่ถูกดัดแปลงอะไรเลย หากไม่มีการ merge เกิดขึ้น ดังนั้นหากเกิดข้อผิดพลาดบน branch ใด ๆ ขึ้น
โค้ดต้นฉบับก็จะยังปลอดภัยอยู่)
