Question1: คิดว่าทําไมถึงยังต้องเรียนเกี่ยวกับ jQuery ทั้งที่ปัจจุบัน Javascript ปกติก็เขียนได้ง่ายขึ้น และมีตัวช่วยอื่นมากขึ้น แถม jQuery ยังเป็นต้นเหตุที่ทําให้เว็บช้าลงด้วย

Answer: เพราะว่าปัจจุบันประเทศไทยยังมีองค์กรและเว็บไซต์ที่นิยมการใช้ jQuery อยู่เยอะมาก ๆ (อ้างอิงจากสถิติในคลาสเรียน)
รวมถึง platform wordpress และ e-commerce อื่น ๆ jquery ก็ยังจำเป็นที่จะต้องใช้ในการพัฒนา
ซึ่งความนิยมในการใช้ก็ไม่น่าจะเปลี่ยนแปลงได้เร็วนัก เนื่องด้วยเหตุผลต่าง ๆ
ทำให้เราจะเป็นที่จะต้องเรียนรู้ jQuery เอาไว้ เพื่อโอกาสในการทำงานหรือพัฒนาโปรเจคจากเว็บที่มี jQuery
อีกทั้งสามารถเพิ่มมูลค่าทักษะเราได้อีกด้วย เพราะ developer รุ่นใหม่ ๆ อาจไม่ค่อยสนใจเรียน jQuery กันแล้ว
ทำให้จำนวน developer ที่สามารถทำ jQuery ได้น้อยลงไปเรื่อย ๆ (แสดงว่าเราก็เป็นที่ต้องการมากยิ่งขึ้น)
